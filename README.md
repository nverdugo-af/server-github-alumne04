# Servidor al Núvol amb GitHub – alumne04

Aquest projecte consisteix en la creació i configuració d’un servidor Linux al núvol utilitzant GitHub Codespaces.  
Inclou:

- Accés remot mitjançant SSH  
- Instal·lació i configuració d’Apache  
- Web pública  
- Web privada amb autenticació bàsica  

El repositori conté l’estructura bàsica del projecte i els fitxers necessaris per al desplegament.
